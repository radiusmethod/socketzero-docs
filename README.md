# SocketZero Documentation

Official documentation for SocketZero - secure remote access made simple.

## What is SocketZero?

SocketZero allows you to securely connect to services running in remote networks from your local machine. Think of it as a VPN alternative that's faster, more secure, and doesn't require network-wide access.

## Key Features

- **Zero Trust Security** - Every connection is authenticated and authorized
- **No VPN Required** - Direct tunnels to specific services, not entire networks
- **Easy to Use** - Simple desktop app for Mac, Windows, and Linux
- **Fast** - Lightweight protocol optimized for low latency

## Getting Started

1. **Download** the SocketZero client for your platform
2. **Add a profile** pointing to your organization's SocketZero receiver
3. **Connect** and access your remote services

See the [Quick Start Guide](docs/guides/quick-start.md) for detailed instructions.
